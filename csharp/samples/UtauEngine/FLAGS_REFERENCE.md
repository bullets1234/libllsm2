# L2R - LLSM2 Resampler フラグリファレンス

L2R (LLSM2 Resampler) で使用可能なフラグの完全なリファレンスです。

Inspired by moresampler, L2R brings LLSM2 vocoder technology to UTAU.

## 基本フラグ

### **Pフラグ** - ピッチ検出バイパス

**書式**: `P` (値なし)

**説明**: .frqファイルを無視して、PYIN/LLSMの自動ピッチ推定を強制します。

**用途**:
- .frqファイルのピッチが不正確な場合
- 手動でピッチを調整せずに自動推定を使いたい場合

**例**:
```
L2R.exe input.wav output.wav C4 100 P
```

---

### **Eフラグ** - FRQピッチ使用

**書式**: `E` (値なし)

**説明**: .frqファイルのF0値をそのまま使用します（デフォルトはLLSM推定を使用）。

**用途**:
- .frqファイルのピッチを信頼する場合
- 手動調整したピッチを保持したい場合

**例**:
```
L2R.exe input.wav output.wav C4 100 E
```

**注意**: PフラグとEフラグは排他的です。

## 音質調整フラグ

### **Bフラグ** - 息成分の強度（Breathiness）

**書式**: `B<0-100>`

**デフォルト**: `B50`

**説明**: 音声の息っぽさを調整します。

| 値 | 効果 |
|----|------|
| `B0` | 息成分なし（クリアな声） |
| `B50` | 標準（デフォルト） |
| `B100` | 最大（ささやき声） |

**用途**:
- ささやき声・息声の表現: `B80-B100`
- クリアな声: `B0-B30`
- 自然な声: `B40-B60`

**例**:
```
L2R.exe input.wav output.wav C4 100 B70
```

---

### **gフラグ** - ジェンダーファクター

**書式**: `g<-100〜+100>`

**デフォルト**: `g0`

**説明**: 声のジェンダー特性を調整します。

| 値 | 効果 |
|----|------|
| `g-50` | より女性的な声（フォルマント上昇） |
| `g0` | 変化なし（デフォルト） |
| `g+50` | より男性的な声（フォルマント下降） |

**用途**:
- 声の性別感を変える
- キャラクター性の調整
- フォルマント微調整

**例**:
```
L2R.exe input.wav output.wav C4 100 g-30  # やや女性的
L2R.exe input.wav output.wav C4 100 g+20  # やや男性的
```

**注意**: 極端な値（±50以上）は不自然になる可能性があります。

---

### **Fフラグ** - フォルマント追従率

**書式**: `F<0-100>`

**デフォルト**: `F100`

**説明**: ピッチシフト時にフォルマント（声道共鳴特性）がどれだけF0に追従するかを制御します。

| 値 | 効果 |
|----|------|
| `F0` | フォルマント完全固定（声質保持） |
| `F50` | 50%追従（推奨・自然なバランス） |
| `F100` | 完全追従（デフォルト、声質変化大） |

**用途**:
- ピッチを変えても声質を保つ: `F0-F30`
- 自然なピッチシフト: `F40-F60`
- 完全追従（不自然になりやすい）: `F100`

**例**:
```
L2R.exe input.wav output.wav C5 100 F0   # 高音でも元の声質
L2R.exe input.wav output.wav C3 100 F50  # 低音で自然
```

**推奨値**: 
- ピッチ変化が大きい場合: `F30-F50`
- ピッチ変化が小さい場合: `F70-F100`

## 高度なフラグ

### **Oフラグ** - オーバーサンプリング

**書式**: `O` (値なし)

**説明**: 4倍オーバーサンプリング（176.4kHz）で合成後、44.1kHzにダウンサンプリングします。

**効果**:
- ✅ エイリアシングノイズ削減
- ✅ 高域成分の正確な再現
- ✅ 全体的な音質向上
- ⚠️ 処理時間が約2-3倍増加

**用途**:
- 最高品質が必要な場合
- 高音域の原音（C5以上）
- マスタリング用途

**例**:
```
L2R.exe input.wav output.wav C4 100 O
L2R.exe input.wav output.wav C5 100 OB60F50  # 他のフラグと併用
```

---

### **Hフラグ** - High-Resolution Mode

**書式**: `H` (値なし)

**説明**: F0に応じて倍音数を動的に調整し、20kHzまでの帯域をカバーします。

| 状態 | 動作 |
|------|------|
| **指定なし** | maxnhar=800固定（標準品質） |
| **H** | maxnhar動的調整（100-2000、F0依存） |

**効果**:
- ✅ 高音域での倍音不足解消
- ✅ 20kHzまでの周波数カバレッジ保証
- ✅ F0に応じた最適な倍音数
- ⚠️ 処理時間が約10-30%増加

**用途**:
- 高音域の原音（C5以上）
- 最高品質が必要な場合
- 倍音豊富な音色

**例**:
```
L2R.exe input.wav output.wav C5 100 H  # 高音域で高品質
L2R.exe input.wav output.wav C4 100 HOB50F50  # 他のフラグと併用
```

---

### **Rフラグ** - RPS (Repeated Phase Synchronization)

**書式**: `R` (値なし)

**説明**: 補間時の位相同期処理を有効化し、周期的なアーティファクトを削減します。

| 状態 | 動作 |
|------|------|
| **指定なし** | 通常の位相補間（デフォルト） |
| **R** | RPS有効（位相同期処理） |

**効果**:
- ✅ 子音速度変更時の「ウワウワ」トレモロ削減
- ✅ 周期的な位相アーティファクト削減
- ✅ より滑らかな音質

**用途**:
- 子音速度を大きく変更する場合
- ピッチ変化が激しい場合
- 周期的なノイズが気になる場合

**例**:
```
L2R.exe input.wav output.wav C4 100 R  # RPS有効
```

---

### **Qフラグ** - F0 Quality Refinement（F0精密化）

**書式**: `Q` (値なし)

**デフォルト**: オフ（未指定）

**説明**: LLSM2の高調波解析ベースのF0リファインメント機能を有効化します。

**効果**:
- ✅ ビブラートの滑らかさ向上
- ✅ ポルタメントの自然さ改善
- ✅ 子音→母音遷移での音程安定性向上
- ✅ 分析・合成段階でのF0ノイズ低減

**推奨**: ビブラート・ポルタメント多用の曲で効果大

**例**:
```
L2R.exe input.wav output.wav C4 100 Q
```

---

### **Wフラグ** - Adaptive Window Size（適応的窓サイズ）

**書式**: `W` (値なし)

**デフォルト**: オフ（固定4.0周期）

**説明**: F0に応じて分析窓サイズを動的に調整します。

**調整ロジック**:
- **低音域 (F0<120Hz)**: rel_winsize=3.0 → 時間分解能向上
- **中音域**: rel_winsize=4.0 → 標準バランス
- **高音域 (F0>350Hz)**: rel_winsize=5.0 → 周波数分解能向上

**効果**:
- ✅ 低音での子音明瞭度向上（窓が狭まり速い変化に追従）
- ✅ 高音での倍音豊かさ向上（窓が広がり詳細な周波数解析）
- ✅ 滑舌・発音の自然さ改善

**推奨**: 低音男声（<150Hz）、高音女声（>350Hz）で特に効果大

**注意**: 計算コスト約10-20%増加

**例**:
```
L2R.exe input.wav output.wav C4 100 W
```

---

### **Uフラグ** - Unvoiced Attenuation（無声音減衰）

**書式**: `U<6-20>`

**デフォルト**: オフ（未指定）

**説明**: F0=0フレーム（無声音）のVTMAGN（声道スペクトル）を減衰させ、無声子音の明瞭度を向上します。

**推奨値**:
- `U9`: -9dB減衰（標準、推奨）
- `U12`: -12dB減衰（強調）
- `U15`: -15dB減衰（最大明瞭度）

**効果**:
- ✅ さ行/ざ行の区別明確化（無声/有声の分離）
- ✅ 破裂音（た行/だ行）のアタック鋭角化
- ✅ 無声子音連続時の個別性向上
- ✅ 息継ぎ・ブレスの自然化

**メカニズム**: Layer1分析時に無声部に有声音のスペクトル特性が漏れる現象を補正

**推奨**: さ行・た行が不明瞭な音源、子音明瞭度重視の用途

**例**:
```
L2R.exe input.wav output.wav C4 100 U9   # 標準
L2R.exe input.wav output.wav C4 100 U12  # 強調
```

---

### **Tフラグ** - Spectral Tilt（スペクトル傾斜）

**書式**: `T<±0-12>`

**デフォルト**: `T0`（フラット）

**説明**: 周波数全域でスペクトル傾斜を調整し、声質の明るさ/暗さを制御します。

**原理**: VTMAGNに対して周波数依存のゲイン調整（dB/octave）を適用

**推奨値**:

| 値 | 効果 | 用途 |
|----|------|------|
| `T-12` | 超暗い声（高域-12dB/oct） | 重低音、ダークな雰囲気 |
| `T-6` | 暗い声（高域-6dB/oct） | 柔らかい声質、バラード |
| `T0` | フラット（標準） | **デフォルト推奨** |
| `T+3` | やや明るい | ポップス、クリアな発音 |
| `T+6` | 明るい声（高域+6dB/oct） | アニメ声、明瞭度重視 |
| `T+12` | 超明るい声（高域+12dB/oct） | 強調、特殊効果 |

**効果**:
- ✅ 声質の明るさ/暗さを直感的に調整
- ✅ ウィスパー（T-10）～叫び声（T+10）の表現
- ✅ 既存音源の音質補正に有効

**計算式**: `gain(f) = tilt × log₂(f[kHz])`

**他フラグとの組み合わせ**:
- `g-12 T+6`: 女性化 + 明るい声質
- `B80 T-6`: 息声 + 柔らかい声
- `U9 T+3`: 子音明瞭 + クリアな発音

**注意**: 過度な値（±10以上）は不自然になる可能性あり

**例**:
```
L2R.exe input.wav output.wav C4 100 T+6   # 明るい声
L2R.exe input.wav output.wav C4 100 T-6   # 暗い声
L2R.exe input.wav output.wav C4 100 T0    # フラット（標準）
```

---

### **M+フラグ** - Modulation Boundary Fade

**書式**: `M+` (値なし)

**説明**: モジュレーション境界でのフェード処理を有効化し、境界アーティファクトを削減します。

**効果**:
- ✅ ノート境界でのクリック音削減
- ✅ より滑らかな接続

**用途**:
- ノート接続が気になる場合
- 境界でのアーティファクトを削減したい場合

**例**:
```
L2R.exe input.wav output.wav C4 100 M+
```

---

### **M1フラグ** - Alternative Modulation Mode

**書式**: `M1` (値なし)

**説明**: 代替モジュレーションモードを使用します。

**注意**: 実験的機能です。

**例**:
```
L2R.exe input.wav output.wav C4 100 M1
```

## 実験的フラグ（非推奨）

### **Nフラグ** - Neural Vocoder Blend（非推奨）

**書式**: `N<0-100>`

**デフォルト**: `N0` (オフ)

**警告**: ⚠️ **このフラグは非推奨です。使用を推奨しません。**

**説明**: HiFi-GAN Neural VocoderによるLLSM出力とのブレンド機能。

**非推奨の理由**:
- 品質が不安定
- 処理時間が大幅に増加
- GPU必須
- モデルファイルの配布/管理が煩雑

**注意**: 互換性のため機能は残されていますが、使用は推奨しません。

## フラグの組み合わせ

複数のフラグは連結して指定できます：

```bash
# 高品質セット（推奨）
L2R.exe input.wav output.wav C4 100 HRF50

# 最高品質セット
L2R.exe input.wav output.wav C4 100 HROF50

# F0精密化 + 高品質（ビブラート曲に推奨）
L2R.exe input.wav output.wav C4 100 HRQF50

# 適応的窓 + 高品質（低音男声・高音女声に推奨）
L2R.exe input.wav output.wav C4 100 HRWF50

# 無声音明瞭化 + 高品質（子音重視）
L2R.exe input.wav output.wav C4 100 HRU9F50

# スペクトル傾斜 + 高品質（明るい声質）
L2R.exe input.wav output.wav C4 100 HRT+6F50

# フル品質セット（Q+W+U+T+R）
L2R.exe input.wav output.wav C4 100 HRQWU9T+3RF50

# 超高品質セット（全機能有効）
L2R.exe input.wav output.wav C4 100 HROQWU9T+6RF50

# 息声 + フォルマント固定 + 高解像度
L2R.exe input.wav output.wav C4 100 B80F0H

# 女性的 + 明るい声質 + RPS
L2R.exe input.wav output.wav C4 100 g-30T+6F50R

# ピッチ自動検出 + 標準設定
L2R.exe input.wav output.wav C4 100 P
```

## 使用例

### 1. 標準的な使用（推奨）
```bash
L2R.exe input.wav output.wav C4 100 RF50
```
- RPS有効（アーティファクト削減）
- フォルマント50%追従（自然な音質）

### 2. 高品質レンダリング（推奨）
```bash
L2R.exe input.wav output.wav C4 100 HRF50
```
- 高解像度モード
- RPS有効
- 自然なフォルマント追従

### 3. 最高品質レンダリング
```bash
L2R.exe input.wav output.wav C4 100 HROF50
```
- 高解像度モード
- RPS有効
- オーバーサンプリング
- フォルマント50%追従
- 高解像度モード
- RPS有効
- オーバーサンプリング
- 自然なフォルマント追従

### 4. ささやき声
```bash
L2R.exe input.wav output.wav C4 80 B90RF50
```
- velocity 80（やや静か）
- 息成分90%
- RPS + フォルマント調整

### 5. 高音での声質保持
```bash
L2R.exe input.wav output.wav C5 100 HF20R
```
- 高解像度モード（高音域で重要）
- フォルマント20%追従（ほぼ固定）
- RPS有効

### 6. 低音での自然な響き
```bash
L2R.exe input.wav output.wav C3 100 F60g+10R
```
- フォルマント60%追従
- やや男性的な響き
- RPS有効

### 7. キャラクター性の調整
```bash
# 女性的なキャラクター
L2R.exe input.wav output.wav C4 100 g-40B45F40HR

# 男性的なキャラクター
L2R.exe input.wav output.wav C4 100 g+30B55F50R
```

## デフォルト値一覧

| フラグ | デフォルト値 | 推奨設定 |
|--------|-------------|----------|

## バージョン履歴

### 現在のバージョン (L2R)
- NFFT: 8192（高周波数分解能）
- maxnhar: 800（標準）または動的（Hフラグ：100-2000）
- npsd: 128（ノイズ品質改善）
- maxnhar_e: 5（標準）または動的（Hフラグ：5-10）
- 補間: Monotonic Cubic（Fritsch-Carlson）
- NM補間: 完全実装（PSD + EDC + eenv）
- RPS: Frame-level実装
- Neural Vocoder: HiFi-GAN統合
- Hフラグ: F0依存動的倍音調整（20kHzカバレッジ）
