## 基本フラグ

### **Pフラグ** - ピッチ検出バイパス

**書式**: `P` (値なし)

**説明**: .frqファイルを無視して、PYIN/LLSMの自動ピッチ推定を強制します。

**用途**:
- .frqファイルのピッチが不正確な場合
- 手動でピッチを調整せずに自動推定を使いたい場合

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 P
```

---

### **Eフラグ** - FRQピッチ使用

**書式**: `E` (値なし)

**説明**: .frqファイルのF0値をそのまま使用します（デフォルトはLLSM推定を使用）。

**用途**:
- .frqファイルのピッチを信頼する場合
- 手動調整したピッチを保持したい場合

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 E
```

**注意**: PフラグとEフラグは排他的です。

## 音質調整フラグ

### **Bフラグ** - 息成分の強度（Breathiness）

**書式**: `B<0-100>`

**デフォルト**: `B50`

**説明**: 音声の息っぽさを調整します。

| 値 | 効果 |
|----|------|
| `B0` | 息成分なし（クリアな声） |
| `B50` | 標準（デフォルト） |
| `B100` | 最大（ささやき声） |

**用途**:
- ささやき声・息声の表現: `B80-B100`
- クリアな声: `B0-B30`
- 自然な声: `B40-B60`

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 B70
```

---

### **gフラグ** - ジェンダーファクター

**書式**: `g<-100〜+100>`

**デフォルト**: `g0`

**説明**: 声のジェンダー特性を調整します。

| 値 | 効果 |
|----|------|
| `g-50` | より女性的な声（フォルマント上昇） |
| `g0` | 変化なし（デフォルト） |
| `g+50` | より男性的な声（フォルマント下降） |

**用途**:
- 声の性別感を変える
- キャラクター性の調整
- フォルマント微調整

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 g-30  # やや女性的
UtauEngine.exe input.wav output.wav C4 100 g+20  # やや男性的
```

**注意**: 極端な値（±50以上）は不自然になる可能性があります。

---

### **Fフラグ** - フォルマント追従率

**書式**: `F<0-100>`

**デフォルト**: `F100`

**説明**: ピッチシフト時にフォルマント（声道共鳴特性）がどれだけF0に追従するかを制御します。

| 値 | 効果 |
|----|------|
| `F0` | フォルマント完全固定（声質保持） |
| `F50` | 50%追従（推奨・自然なバランス） |
| `F100` | 完全追従（デフォルト、声質変化大） |

**用途**:
- ピッチを変えても声質を保つ: `F0-F30`
- 自然なピッチシフト: `F40-F60`
- 完全追従（不自然になりやすい）: `F100`

**例**:
```
UtauEngine.exe input.wav output.wav C5 100 F0   # 高音でも元の声質
UtauEngine.exe input.wav output.wav C3 100 F50  # 低音で自然
```

**推奨値**: 
- ピッチ変化が大きい場合: `F30-F50`
- ピッチ変化が小さい場合: `F70-F100`

## 高度なフラグ

### **Oフラグ** - オーバーサンプリング

**書式**: `O` (値なし)

**説明**: 4倍オーバーサンプリング（176.4kHz）で合成後、44.1kHzにダウンサンプリングします。

**効果**:
- ✅ エイリアシングノイズ削減
- ✅ 高域成分の正確な再現
- ✅ 全体的な音質向上
- ⚠️ 処理時間が約2-3倍増加

**用途**:
- 最高品質が必要な場合
- 高音域の原音（C5以上）
- マスタリング用途

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 O
UtauEngine.exe input.wav output.wav C5 100 OB60F50  # 他のフラグと併用
```

---

### **Rフラグ** - 残差ノイズコピー

**書式**: `R` (値なし)

**説明**: ノイズ成分（PSDRES）を近傍フレームからランダムコピーします（demo-stretch.c互換動作）。

| 状態 | 動作 |
|------|------|
| **指定なし** | 補間されたPSDRESを使用（デフォルト） |
| **R** | ランダムPSDRESコピー有効 |

**用途**:
- ノイズの時間変動を増やしたい場合
- demo-stretch.cとの互換性が必要な場合

**注意**: 
- ⚠️ **長時間ストレッチ（3倍以上）で末尾がノイジーになる可能性**
- 通常は指定不要（デフォルトで問題なし）

**例**:
```
UtauEngine.exe input.wav output.wav C4 100 R  # ランダムコピー有効
```

## フラグの組み合わせ

複数のフラグは連結して指定できます：

```bash
# 息声 + フォルマント固定 + オーバーサンプリング
UtauEngine.exe input.wav output.wav C4 100 B80F0O

# 女性的 + 自然なフォルマント + 高品質
UtauEngine.exe input.wav output.wav C4 100 g-30F50O

# ピッチ自動検出 + 標準設定
UtauEngine.exe input.wav output.wav C4 100 P
```

## 使用例

### 1. 標準的な使用（フラグなし）
```bash
UtauEngine.exe input.wav output.wav C4 100
```

### 2. 高品質レンダリング
```bash
UtauEngine.exe input.wav output.wav C4 100 OB50F50
```
- オーバーサンプリング
- 標準的な息成分
- 自然なフォルマント追従

### 3. ささやき声
```bash
UtauEngine.exe input.wav output.wav C4 80 B90
```
- velocity 80（やや静か）
- 息成分90%

### 4. 高音での声質保持
```bash
UtauEngine.exe input.wav output.wav C5 100 F20O
```
- フォルマント20%追従（ほぼ固定）
- オーバーサンプリングで高品質

### 5. 低音での自然な響き
```bash
UtauEngine.exe input.wav output.wav C3 100 F60g+10
```
- フォルマント60%追従
- やや男性的な響き

### 6. キャラクター性の調整
```bash
# 女性的なキャラクター
UtauEngine.exe input.wav output.wav C4 100 g-40B45F40

# 男性的なキャラクター
UtauEngine.exe input.wav output.wav C4 100 g+30B55F50
```

## トラブルシューティング

### 声質が不自然に変わる
- **原因**: Fフラグがデフォルト（F100）
- **解決**: `F50`または`F30`を指定

### 高音が細く・子供っぽくなる
- **原因**: フォルマントが完全追従している
- **解決**: `F0`〜`F30`でフォルマント固定

### 低音が太すぎる・不自然
- **原因**: フォルマント固定しすぎ
- **解決**: `F50`〜`F70`で適度に追従

### 長いノートの末尾がノイジー
- **原因**: Rフラグが有効になっている
- **解決**: Rフラグを外す（デフォルトで解決）

### 音がこもる
- **原因**: 現在のバージョンでは改善済み（NFFT=8192, maxnhar=800）
- **解決**: Oフラグで追加改善可能
