■ コマンドライン引数（UTAU標準）
--------------------------------------------------------------------------------
  args[3]   velocity      int    デフォルト: 100
  args[9]   volume        int    デフォルト: 100
  args[10]  modulation    int    デフォルト: 0

■ フラグ（args[4] 文字列から解析）

  【分析制御】
--------------------------------------------------------------------------------
  フラグ  型    デフォルト  範囲      大小文字  説明
--------------------------------------------------------------------------------
  P       bool  false      -         不問      .frq無視、PYIN自動推定を強制
  H       bool  false      -         不問      動的倍音数調整（F0に応じてナイキストまでカバー）
  Q       bool  false      -         不問      高調波解析ベースのF0精密化
  W       bool  false      -         不問      F0に応じた分析窓サイズ動的調整
  E       bool  false      -         不問      FRQのF0をそのまま使用

  【声質制御（数値）】
--------------------------------------------------------------------------------
  フラグ  型    デフォルト  範囲      大小文字    パターン        説明
--------------------------------------------------------------------------------
  B       int   50         0-100     不問        B(\d+)          息成分強度
  g       int   0          制限なし  小文字のみ  g([+-]?\d+)     ジェンダー（+男性的、-女性的）
  F       int   100        0-100     不問        F(\d+)          フォルマント追従率
  T       int   0          -12～+12  不問        T([+-]?\d+)     テンション
  K       int   50         0-100     不問        K(\d+)          声門閉鎖係数（0=息漏れ、50=標準、100=硬い声）
  G       int   0          1-100     大文字のみ  G(\d+)          グロウル効果（PBP合成）
  U       int   0          6-20      不問        U(\d+)          無声音減衰 dB
  D       int   0          1-100     大文字のみ  D(\d+)          HNR改善（ノイズ低減強度）
  C       int   0          1-100     大文字のみ  C(\d+)          子音原音PCMブレンド率

  【合成制御（トグル）】
--------------------------------------------------------------------------------
  フラグ  型    デフォルト  大小文字             説明
--------------------------------------------------------------------------------
  L       bool  false      不問                 声門パラメータ自動推定
  X       bool  false      不問                 振幅補正を固定比率に切替
  O       bool  false      不問                 4倍オーバーサンプリング合成(非推奨)
  M+/M1   bool  false      M+:完全一致/M1:不問  ノート境界modulationフェード

  【ストレッチ制御(※非推奨)】
--------------------------------------------------------------------------------
  フラグ  型    デフォルト  大小文字  説明
--------------------------------------------------------------------------------
  e       bool  false      不問      トランジェント保護タイムストレッチ
  A       bool  false      不問      ピッチマーク駆動ストレッチ
  V       bool  false      不問      F0ベース子音境界自動修正
